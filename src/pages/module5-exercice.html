Exercice 5 : La Vue et les Templates

Associez le bloc de données au bon template de la Vue.

Données du Contrôleur

$user = ['name' => 'Lancelot', 'level' => 15];
$products = [['name' => 'Épée'], ['name' => 'Bouclier']];
$error = ['message' => 'Accès interdit'];

Templates de la Vue

<ul> <?php foreach ($products as $item): ?> <li><?php echo htmlspecialchars($item['name']); ?></li> <?php endforeach; ?> </ul>
<div class="error"> <?php echo htmlspecialchars($error['message']); ?> </div>
<h1>Profil de <?php echo htmlspecialchars($user['name']); ?></h1> <p>Niveau : <?php echo htmlspecialchars($user['level']); ?></p>
